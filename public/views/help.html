What is turf?

According to the turf github page:

Turf is a JavaScript library for spatial analysis. It includes traditional spatial operations, helper functions for creating GeoJSON data, and data classification and statistics tools. Turf can be added to your website as a client-side plugin, or you can run Turf server-side with Node.js

What is turfjs.party?

A project designed to let you play around and learn turf in the browser without worrying about setting up the boilerplate code to get started.

Accessing Geometries

Geometries can be accessed in the Editor environment by using the g object. If you look at the Geometries tab, you can view and change the name of any feature you've drawn or added.

Adding Geometries in the Editor

Geometries can be added in the Editor either by using the g object or by using //= comments: 

var feature1 = // some geojson
g.feature1 = feature1;
is the same as 
var feature1 = // some geojson
//=feature1

Styling Geometries

Geometries will be styled using Leaflet's style options. To set styles in the Editor, add leaflet style options to a feature's properties: 

g.feature1.properties.color = "#ff00ff";
g.feature1.properties.weight = 2;

Time Based Functions

The Editor knows how to use both setInterval and setTimeout, so you can do pretty fun things: 

var pt = turf.point([-85, 45]);

var buff = 1;
var inc = 1;

map.setZoom(9);

setInterval(function () {
    var ft = turf.buffer(pt, buff, 'miles');

    buff += inc
    if (buff > 10) inc = -1;
    if (buff < 2) inc = 1;

    //=ft
}, 50);
