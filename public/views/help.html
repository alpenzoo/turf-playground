Accessing Geometries

Geometries can be accessed in the Editor environment by using the g object. If you look at the Geometries tab, you can view and change the name of any feature you've drawn or added.

Styling Geometries

Geometries will be styled using Leaflet's style options. To set styles in the Editor, add leaflet style options to a feature's properties: 
g.feature1.properties.color = "#ff00ff";
g.feature1.properties.weight = 2;
